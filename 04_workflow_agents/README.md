# 04_workflow_agents

官方教程[text](https://docs.langchain.com/oss/python/langgraph/workflows-agents)

## 核心要点

官方教程里面举了许多例子。

- 提示链条（就是一条链条到结束）
- 并行化，多个任务并行执行（静态，明确定义子任务），后续节点汇聚结果
- 路由，交给不同的来执行
- 协调者工作者，拆分子任务（动态），后续节点汇聚结果
- 评估器优化器，带一个小循环
- agents，也是带小循环

其实大体都差不多，后续再快速过一下，目前看来只有协调者工作者会有新内容（send）。

然后每个例子都给出了两种实现，一种是显式构建图，一种是用装饰器+python函数的写法。通常大项目还是要用前者，后者无法直接绘制图的结构。
